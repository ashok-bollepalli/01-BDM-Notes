# Java Fullstack Leads Handling Points

## Course Overview

- Java Fullstack is a job-oriented course for freshers, final-year students, job seekers, and working professionals.
- Students will learn backend, frontend, database, REST APIs, projects, and deployment basics.
- Goal: prepare students for Java Fullstack Developer roles.

## Who Can Join

- B.Tech, MCA, BCA, B.Sc, Degree students
- Freshers and job seekers
- Non-IT students interested in software jobs
- Working professionals planning for career switch
- Basic computer knowledge is enough

## Pre-Requisites

- No advanced coding knowledge required
- Basic computer knowledge is enough
- Laptop/Desktop and internet required
- Daily practice is important

## Course Content

- Core Java
- Advanced Java
- JDBC, Servlets, JSP
- SQL and MySQL
- Spring Framework
- Spring Boot
- REST APIs
- Spring Data JPA / Hibernate
- Microservices basics
- Spring Security and JWT
- HTML, CSS, Bootstrap, JavaScript
- Angular / React JS
- Git, GitHub, Maven, Postman, Swagger
- Real-time project development

## Services We Offer

- Daily live classes
- Backup videos for revision
- Soft copy materials
- Doubt clarification
- Real-time projects
- LMS access for notes/videos
- Interview preparation
- Resume guidance
- Mock interview support
- Placement assistance
- Course completion certificate

## Important Points to Tell Leads

- Course starts from basics.
- Non-IT students can also join.
- Regular attendance and daily practice are required.
- Projects help in resume and interviews.
- We provide placement assistance, not guaranteed job.
- Student effort is very important for job success.

## Common Lead Questions

### Can I join without coding knowledge?

Yes, we start from Core Java basics.

### Do you provide recordings?

Yes, backup videos are provided.

### Do you provide projects?

Yes, real-time projects are included.

### Do you provide placement?

Yes, placement assistance, resume guidance, job updates, and interview preparation are provided.

### How much practice is required?

Minimum 2 to 3 hours daily practice is recommended.

## BDM Follow-Up Points

- Understand student background first.
- Ask their goal: job, career switch, or skill upgrade.
- Explain course based on their requirement.
- Share batch timing, fee, LMS, Zoom, and payment details clearly.
- Maintain proper lead status and follow-up date.
- Add clear comments after every call.
- Do not promise guaranteed job.
